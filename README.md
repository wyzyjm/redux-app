# 项目构建
1. `npx create-react-app redux-app`
2. 删除多余文件
3. 编写文件
<!-- 4. `npm i redux` | `yarn add redux` -->
4. `npm install @reduxjs/toolkit` | `yarn add @reduxjs/toolkit`
5. `yarn add react-redux`
6. `npm start` | `yarn start`


## 编写过程
1. 创建 store
2. 添加到 React中
3. 创建 切片 / action 函数
4. 将切片添加到 store中 reducer中
5. 组件中使用 store
